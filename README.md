# PDF Sentence Navigator

## Install
1. Open Chrome and go to `chrome://extensions`.
2. Enable Developer mode.
3. Click Load unpacked.
4. Select the `PDF-Sentence-Navigator` folder.

## Use
1. Open a PDF in Chrome.
2. Click the extension icon once to activate it on the current tab.
3. Use Tab and Shift+Tab to move through sentences.

## Note
This version avoids the invalid `content_scripts.matches` pattern and loads via the service worker + injected script approach.
